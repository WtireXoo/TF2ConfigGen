# TF2ConfigGen
<img width="747" height="280" alt="text-effect-1762521556857" src="https://github.com/user-attachments/assets/76f8038f-cb8f-4106-bf2e-89ce2fd5641a" />

- A free, *closed* source program that creates a personal Team Fortress 2 Config for you.
- No administrative permissions needed.
- Every single script that has been took from a person have been creddited in the program.

# What it stores

- This program stores your computer name, major, build number, platform id, and csd version.
- Please know that this is information that is stored is only on *your computer*.

# Is it a malware?
- No it is not. A small tool has no reason to put a virus in it.
- Still not sure? Check out the virustotal scan: https://www.virustotal.com/gui/file/2f7f5410169a7c5b0df57d1ad7a242c3ba324683322b4f11230cca87338ab392/detection

# Compatibility
- This tool *should* work on all devices.. There were no reports of issues at all.
- However, i still suggest you to use "Windows 11" or "Windows 10" not any modded versions.
- This tool does **NOT** support linux, it purely works mostly on the windows api in c++.
